# Site Core Hourly Rate as Billing Source

Core childcare pricing is configured as one current positive hourly rate per nursery site, owned by owner-level setup, rather than as an editable rate on each child record. This trades child-specific pricing flexibility for simpler multi-site billing control, clearer owner accountability, and consistent future invoice generation; child-specific discounts or exceptions will be handled separately from the site core hourly rate if needed later.
