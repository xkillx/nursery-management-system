Configuration Tips

  • Use a clear, recognisable name — booking patterns will list it for practitioners every day.
  • Times must be entered in 24-hour format (HH:MM). End time must be after start time.
  • Archived types remain referenced by historical booking patterns and can be reactivated later.

Booking Pattern Guidance
